# first_flutter

Flutterを用いて作成した、初めてのアプリケーションです。  
学習内容のアウトプットとして制作しました。

---

## 📱 アプリ概要

本アプリは、ランダムな色を生成し、お気に入り登録・管理ができるシンプルなアプリです。

---

## ✨ 主な機能

### ■ Colorsページ
- ランダムなカラーを10個生成
- 各カラーに対して「Like（お気に入り）」が可能

### ■ Favoriteページ
- お気に入り登録したカラー一覧を表示
- 不要なカラーを削除（Delete）可能

## 🚀 実行方法（ローカル環境）

以下の手順でアプリをローカルで実行できます。

### ① リポジトリをクローン

git clone https://github.com/nightree54/first_flutter.git  
cd first_flutter

### ② 依存関係をインストール

flutter pub get

### ③ アプリを起動

flutter run

※ エミュレーターまたは実機を事前に起動してください。
